# Blog Web App
A blog website made using Django. The admin can create new posts, edit posts, delete posts and publish drafts. Others can view the posts and comment.

Check out the blog here: https://ransh.pythonanywhere.com

## Technologies used
1. Django
1. HTML
1. CSS
1. Bootstrap
